demo-joc-haproxy
================

haproxy configuration with Jenkins Operations Center

Depends on Jenkins servers running with fixed JNLP ports as follows:

	40001 : JOC
	40002 : API-TEAM
	40003 : WEB-TEAM
	
Also depends on these SSH ports

    40012 : API-TEAM
    40013 : WEB-TEAM

See demo-joc for more details
